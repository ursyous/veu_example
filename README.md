## Run development
> 1. Clone source code
> 2. Run command
> ``` bash
> # Move project directory
> $ cd {path}
>
> # Install dependencies packages
> $ npm install
>
> # run application for development
> $ npm run dev
> ```
> 3. See the next
> http://localhost:3000

## Production build

``` bash
# ES Lint execute
$ npm run lint

# install dependencies
$ npm install

# build for production and launch server
$ npm run build
$ npm start
```
